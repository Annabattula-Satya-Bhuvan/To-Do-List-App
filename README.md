# 📝 To-Do List App

A simple **Console-Based To-Do List Application** developed using **Python**. This project allows users to manage their daily tasks through a menu-driven interface. A web version built with **HTML** is also included so the project can be hosted using GitHub Pages.

---

## 📌 Project Description

The To-Do List App helps users organize tasks by allowing them to:

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Exit the application

The project demonstrates the use of Python lists, loops, conditional statements, dictionaries, and user input.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
